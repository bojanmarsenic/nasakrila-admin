# Nasakrila Admin

Administrativna aplikacija za upravljanje paraglajding takmičenjima i događajima.

## Funkcionalnosti

- Upravljanje pilotima
- Upravljanje timovima
- Obrada rezultata
- Dodela startnih brojeva
- CSV import podataka
- Reset rezultata i administrativne akcije
- Prilagođene PHP funkcionalnosti u WordPress okruženju

## Tehnologije

- PHP
- WordPress
- HTML
- CSS
- JavaScript
- CSV obrada podataka

## Opis projekta

Razvoj i prilagođavanje administrativnog sistema za organizaciju takmičenja, obradu podataka i upravljanje učesnicima.
